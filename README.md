# 🌅 Resaca y Horizonte


Proyecto grupal — curso de Videojuegos y Aplicaciones Móviles 2026. Desarrollado con el motor **[Ren'Py](https://www.renpy.org/)**.

---
## 1. Descargar e instalar el Ren'Py 

1. Ir a la página oficial de descargas: **https://www.renpy.org/latest.html**
2. Descargar el instalador correspondiente al sistema operativo 
3. Extraer el SDK en una carpeta de fácil acceso 
---

## 2. Descargar este repositorio



** — Clonar con Git:**
```bash
git clone https://github.com/ahimelevquispe-del/Proyecto_Videojuegos_2026_Grupal.git
```

---

## 3. Ubicar los archivos dentro de un proyecto Ren'Py
1. Abrir el **Ren'Py Launcher**: ejecutar `renpy.sh` (Linux/Mac) o `renpy.exe` (Windows) dentro de la carpeta del SDK.
2. Clic en **"Create New Project"** (Crear nuevo proyecto) y ponerle un nombre, por ejemplo `ResacaYHorizonte`. Esto genera una carpeta nueva con una subcarpeta `game/` ya predeterminada.
3. Entrar a esa carpeta recién creada (algo como `~/RenPy/projects/ResacaYHorizonte/game/`) y **borrar los archivos de ejemplo** que trae por defecto.
4. Copiar **todo el contenido** de este repositorio (`script.rpy`, `options.rpy`, `gui.rpy`, `screens.rpy`, `recursos.rpy`, `audio/`, `images/`, `gui/`, `tl/`, `libs/`) directamente dentro de esa carpeta `game/`.

> Los archivos deben quedar **directamente dentro de `game/`**, sin una subcarpeta intermedia. Es decir: `game/script.rpy`

---
## 4. Ejecutar el juego

1. En el Ren'Py Launcher, seleccionar el proyecto `ResacaYHorizonte` en la lista de proyectos.
2. Clic en **"Launch Project"** (Iniciar proyecto).
3. Finalmente el juego abre en una ventana propia.
